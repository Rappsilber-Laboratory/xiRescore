xiRescore Documentation
======================================

XiRescore is a cross-link mass spectrometry rescoring tool developed by Rappsilber Laboratory.
Low FDR samples from the input data are used to train multiple classifiers using k-fold cross-validation.
These classifiers are then used to calculate new scored for the input samples.
